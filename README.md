# Gmail Compose Function – Manual Test Cases

## Overview
This repository contains manual test cases for the **Compose** feature of Gmail, covering both Traditional and BDD (Behavior-Driven Development) style test formats.

## Assignment Details
- **Module Tested:** Gmail – Compose Function
- **Testing Type:** Manual Testing
- **Test Formats:** Traditional + BDD (Gherkin)
- **Tool Used:** Microsoft Excel

## File
| File Name | Description |
|---|---|
| `Gmail_Compose_TestCases.xlsx` | Complete test case document with 3 sheets |

## Sheet Breakdown
| Sheet | Content |
|---|---|
| Traditional Test Cases | 20 TCs – Positive & Negative with steps, expected results, priority & severity |
| BDD Test Cases | 18 TCs – Gherkin format (Given / When / Then / And) |
| Test Summary | Dashboard showing coverage metrics by category |

## Coverage Areas
- Compose Window Launch
- To / CC / BCC Fields
- Subject & Body
- File Attachments
- Text Formatting
- Draft Saving & Resuming
- Contact Auto-suggest
- Hyperlinks
- Validations & Negative Scenarios
